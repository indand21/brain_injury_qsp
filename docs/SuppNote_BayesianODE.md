# Supplementary Note 4: Bayesian MAP Estimation of Patient-Specific ICP/CPP ODE Parameters

**Manuscript**: A Quantitative Systems Pharmacology–Machine Learning Framework for Multi-Domain Outcome Prediction in Neurocritical Care: A Computational Simulation Study

---

## 1. Motivation and Overview

Supplementary Note 1 describes the deterministic ICP/CPP ODE system whose five parameters (AR_index, C₀_brain, R_csf, k_bbb, k_edema_clear) are personalised from admission clinical variables using closed-form formulas. While this deterministic personalisation is sufficient for the primary QSP-ML pipeline, it does not quantify parameter uncertainty — the same ICP time-series is consistent with a range of parameter combinations.

This note describes an extended Bayesian estimation procedure that:
1. Treats the five ODE parameters as random variables with prior distributions reflecting the clinical formulas
2. Fits each patient's ICP trajectory by maximising the posterior (MAP estimation)
3. Quantifies parameter uncertainty via the Laplace approximation (inverse Hessian at the MAP)
4. Extracts 19 Bayesian features (MAP estimates, posterior SDs, 95% CI widths, and 4 composites) per patient

These Bayesian features form the `Clinical+BayesODE` feature set in the extended ML comparison (Supplementary Table 8).

---

## 2. Observed Data

For each patient, synthetic ICP waveforms are generated by solving the ICP/CPP ODE (Supplementary Note 1) with patient-specific parameters and adding simulated respiratory/cardiac noise. The observed data are 28 window-mean ICP values computed at 6-hour intervals:

$$\mathbf{y}_{\text{obs}} = \{y_1, y_2, \ldots, y_{28}\}$$

where y_w is the mean ICP over a 6-hour window centred at evaluation time t_w. The evaluation times are:

$$\mathbf{T}_{\text{eval}} = [3,\ 9,\ 15,\ 21,\ \ldots,\ 159,\ 165]\ \text{hours} \quad (28 \text{ points at 6-h intervals})$$

All n = 2,000 patients undergo MAP estimation. The Bayesian procedure uses the same ODE system regardless of ICP monitoring status in the simulated clinical record, since synthetic ICP waveforms are generated for all patients from their ODE parameters.

---

## 3. Likelihood

Observations are assumed normally distributed around the ODE prediction:

$$y_t \sim \mathcal{N}\!\left(\text{ICP}_{\text{ODE}}(\theta, t),\ \sigma^2\right)$$

with measurement error σ = 2 mmHg (fixed, reflecting clinical ICP monitoring precision).

The log-likelihood is:

$$\ell(\theta) = -\frac{1}{2\sigma^2} \sum_{t=1}^{T} \left(y_t - \text{ICP}_{\text{ODE}}(\theta, t)\right)^2$$

where ICP_ODE(θ, t) is the solution of the ICP/CPP ODE system (Supplementary Note 1, Equations 5–9) evaluated at time t with parameter vector θ = (AR_index, C₀_brain, R_csf, k_bbb, k_edema_clear).

---

## 4. Prior Distributions

Priors are constructed to be weakly informative, centred on the deterministic personalisation formulas from Supplementary Note 1.

| Parameter | Prior distribution | Mean | SD | Rationale |
|---|---|---|---|---|
| AR_index | Uniform(0.05, 0.99) | — | — | Bounded between near-intact and near-fully-impaired autoregulation; no additional regularisation |
| C₀_brain | Normal(0.80, 0.25²) | 0.80 | 0.25 | Centred on literature normal compliance; SD allows ±1 unit physiological range |
| R_csf | Half-Normal(0, 8.0²) | — | 8.0 | Centred at zero with scale = population mean (8.0 mmHg·min/mL); prevents negative values |
| k_bbb | Half-Normal(0, 0.05²) | — | 0.05 | Centred at mean deterministic value; small scale reflects tight biological constraint |
| k_edema_clear | Half-Normal(0, 0.02²) | — | 0.02 | Centred at mean deterministic value |

The log-prior contributions are:

$$\log p(\text{AR\_index}) = 0 \quad \text{(uniform — no contribution to NLP)}$$

$$\log p(C_0) = -\frac{1}{2}\left(\frac{C_0 - 0.80}{0.25}\right)^2$$

$$\log p(R_{\text{csf}}) = -\frac{1}{2}\left(\frac{R_{\text{csf}}}{8.0}\right)^2$$

$$\log p(k_{\text{bbb}}) = -\frac{1}{2}\left(\frac{k_{\text{bbb}}}{0.05}\right)^2$$

$$\log p(k_{\text{ec}}) = -\frac{1}{2}\left(\frac{k_{\text{ec}}}{0.02}\right)^2$$

## 5. Posterior and MAP Estimation

The log-posterior is:

$$\log p(\theta \mid \mathbf{y}) = \ell(\theta) + \log p(\theta) + \text{const}$$

MAP estimation minimises the negative log-posterior (NLP):

$$\hat{\theta}_{\text{MAP}} = \arg\min_\theta \left[ -\ell(\theta) - \log p(\theta) \right]$$

subject to hard bounds:

| Parameter | Lower bound | Upper bound |
|---|---|---|
| AR_index | 0.05 | 0.99 |
| C₀_brain | 0.05 | 2.50 |
<!-- Note: the penalty check in the code enforces 0.05 as the effective lower bound -->
| R_csf | 0.50 | 50.0 |
| k_bbb | 0.001 | 0.60 |
| k_edema_clear | 0.001 | 0.25 |

Hard bounds are enforced via a large penalty (NLP = 10^10) outside the feasible region, making Powell's method effectively constrained.

**Optimiser**: Powell's method (`scipy.optimize.minimize`, method='Powell') with convergence tolerances ftol = 1x10^-5, xtol = 1x10^-5, maximum 500 iterations. Starting point: deterministic personalisation values from the clinical formulas (Supplementary Note 1, Section 2.4).

---

## 6. Posterior Uncertainty — Laplace Approximation

The Laplace approximation treats the posterior as locally Gaussian around the MAP:

$$p(\theta \mid \mathbf{y}) \approx \mathcal{N}\!\left(\hat{\theta}_{\text{MAP}},\ \mathbf{H}^{-1}\right)$$

where H is the Hessian of the NLP evaluated at the MAP:

$$H_{ij} = \frac{\partial^2 (-\log p(\theta \mid \mathbf{y}))}{\partial \theta_i \partial \theta_j} \bigg|_{\theta = \hat{\theta}_{\text{MAP}}}$$

The Hessian is computed numerically using symmetric central differences:

$$H_{ij} \approx \frac{\nabla_i f(\theta + \varepsilon e_j) - \nabla_i f(\theta - \varepsilon e_j)}{2\varepsilon}$$

with step size ε = 2x10^-4. The Hessian is symmetrised as H = (H + H^T)/2 and regularised with a ridge term λI (λ = 10^-4) to ensure numerical invertibility.

The posterior variance for each parameter is the corresponding diagonal element of H^-1:

$$\text{Var}(\theta_i \mid \mathbf{y}) = \left[H^{-1}\right]_{ii}$$

The 95% credible interval width is approximated as:

$$\text{CI}_{95\%} = 2 \times 1.96 \times \sqrt{\text{Var}(\theta_i \mid \mathbf{y})} = 3.92 \times \sigma_i$$

**Fallback**: For patients where H is near-singular (Hessian inversion fails), the prior SD² values are used as posterior variances. Posterior SDs are capped at physiologically meaningful maxima to prevent Hessian-degenerate solutions from inflating uncertainty estimates.

| Parameter | Min SD cap | Max SD cap |
|---|---|---|
| AR_index | 0.002 | 0.450 |
| C₀_brain | 0.005 | 0.500 |
| R_csf | 0.100 | 10.000 |
| k_bbb | 0.0005 | 0.150 |
| k_edema_clear | 0.0002 | 0.050 |

## 7. Features Extracted (19 per patient)

### 7.1 Per-parameter features (5 parameters × 3 = 15 features)

For each parameter θ_i ∈ {AR_index, C₀_brain, R_csf, k_bbb, k_edema_clear}:

| Feature | Formula | Interpretation |
|---|---|---|
| bayes_{param}_mean | θ̂_MAP,i | MAP estimate (personalised parameter value) |
| bayes_{param}_sd | σ_i = √Var(θ_i) | Posterior standard deviation (parameter uncertainty) |
| bayes_{param}_ci95_width | 3.92 × σ_i | 95% credible interval width |

### 7.2 Composite Bayesian features (4 features)

| Feature | Formula | Interpretation |
|---|---|---|
| bayes_param_uncertainty_index | Mean of σ₁…σ₅ | Overall parameter uncertainty (higher = less identifiable from data) |
| bayes_ar_deviation_from_prior | \|AR̂ − 0.50\| / 0.25 | Deviation of MAP AR_index from prior mean in SD units (higher = ICP data strongly informative about autoregulation) |
| bayes_compliance_recovery_ratio | Ĉ₀_brain / R̂_csf | Ratio of compliance to resistance (higher = better fluid absorption relative to stiffness) |
| bayes_bbb_edema_ratio | k̂_bbb / k̂_edema_clear | Ratio of BBB disruption to oedema clearance rates (higher = net oedema formation) |

## 8. Cohort-Level Results

Population distributions of MAP estimates across n=2,000 patients (Supplementary Table 10):

| Parameter | Median MAP (IQR) | Comparison to deterministic |
|---|---|---|
| AR_index | 0.250 (0.133–0.417) | Systematically lower than deterministic (prior shrinkage toward 0.50) |
| C₀_brain | 0.757 (0.687–0.802) | Slightly below prior mean (0.80) — data pull toward lower compliance |
| R_csf | 10.400 (9.970–11.015) | Systematically above deterministic formulas — MAP inflated by prior scale (σ=8.0) |
| k_bbb | 0.011 (0.008–0.016) | Below deterministic values — MAP regularised toward zero by HalfNormal prior |
| k_edema_clear | 0.017 (0.001–0.019) | Near deterministic mean (0.020) |

The wide 95% CI widths for R_csf (median 39.2) indicate poor identifiability of CSF outflow resistance from ICP time-series alone. This is expected, as R_csf affects the long-term ICP setpoint (through the Davson absorption term) but its effect is largely collinear with C₀_brain over the 7-day window. More informative identification would require CSF drainage experiments (Lundberg plateau measurements).

## 9. Comparison with Deterministic QSP Features

In the extended ML comparison (Supplementary Table 8), Clinical+BayesODE was compared to Clinical+DetQSP:

| Outcome | Clinical | +DetQSP | +BayesODE | Interpretation |
|---|---|---|---|---|
| GOSE 12m | 0.409 | 0.408 | 0.410 | BayesODE: marginal gain (+0.001) |
| FIM Total 12m | 0.514 | 0.509 | 0.515 | BayesODE: marginal gain (+0.001) |
| Mortality (AUC) | 0.564 | 0.579 | 0.568 | DetQSP superior (+0.015 vs +0.004) |

The Bayesian features do not improve substantially over deterministic features. This is consistent with the DAG structure (Supplementary Note 3): mechanistic features are descendants of GCS and APACHE II in the DGP, so Bayesian re-estimation — which is also driven by GCS/APACHE-derived simulated ICP — adds only marginal new information beyond the clinical feature set. In real patient data, where ICP is measured rather than simulated, MAP estimation from observed ICP time-series would be expected to provide genuinely non-collinear ODE parameters.

## 10. Implementation Notes

- Language: Python 3.11 (scipy.optimize.minimize, scipy.integrate.solve_ivp)
- Parallelisation: joblib Parallel (n_jobs = cpu_count − 1); ~15–30 minutes for n=2,000 patients on 8-core CPU
- Cached output: `bayes_ode_parameters_n2000.csv` (19 columns × 2,000 rows); reloaded on subsequent runs
- All 2,000 patients undergo MAP estimation using synthetic ICP waveforms generated from their ODE parameters; no distinction is made between ICP-monitored and non-monitored patients in the Bayesian pipeline
